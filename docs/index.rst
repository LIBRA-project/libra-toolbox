libra-toolbox
=============

``libra-toolbox`` is a powerful and flexible library designed to help the design and analysis of LIBRA experiments.
This documentation will guide you through the various features and functionalities of libra-toolbox.


Installation
------------

To install libra-toolbox, use pip:

.. code-block:: bash

   pip install git+https://github.com/LIBRA-project/libra-toolbox


Roadmap
-------
.. role:: strike
    :class: strike

- :strike:`Tritium model: 0D model for tritium release`
- :strike:`Neutron detection: tools and scripts for neutron detection post processing`
- Neutronics: tools to facilitate building OpenMC models
- Tritium transport: tools to facilitate building FESTIM tritium transport models
